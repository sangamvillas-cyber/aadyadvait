Vault setup

Encrypt family-identity.json and every sensitive PDF/JPG in this repo with a random data key. The data key is wrapped under your backup passphrase today; you'll add Touch ID and a YubiKey on the live site after Cloudflare Access is up.

About the passphrase you're about to set:
  • It's a backup-only recovery key. Daily unlock will use Touch ID / YubiKey once enrolled.
  • Print it, put it in a safe deposit box. Don't keep it on your laptop or in a notes app.
  • If you lose every authenticator and the passphrase, all encrypted data is permanently lost. That's the cost of zero-knowledge.
  • This page needs Chrome / Edge / Brave (File System Access API).
